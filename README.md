# risp

A toy Lisp interpreter written in Rust with reference to [Risp (in (Rust) (Lisp))](https://stopa.io/post/222). Not following any Lisp standard.

## Supported Functions

- `defvar`, `setq`

- `if`

- `+`, `-`, `*`, `/`, `mod`, `rem`

- `=`, `!=`, `<`, `>`, `<=`, `>=`

- `max`, `min`
